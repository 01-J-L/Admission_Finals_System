Backup Database

Download a complete SQL dump of the current system database.

This process will generate a .sql file containing all student records, grades, payments, and system settings. It is recommended to perform this action weekly or before making major changes.

Restore Database

Overwrite current data with a previous backup file.

Warning: Critical Action

Restoring a backup will permanently overwrite all current data. Any data added since the backup date will be lost. This action cannot be undone.
